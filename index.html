Math Game!

How the game works:

  1. A number between 0 and 10 will be displayed. Remember this number.
  2. The first number will go away, and a second number will be displayed.
  3. Add the first number to the currant number. Select the number that matches the sum.
  4. Another number will be displayed.
  5. Add the currently displayed number to the number that was immediately preceding it.
  6. The game will run for 60 numbers.
  7. At the game's completion, your score will be displayed.
  8. High score will be displayed, and a new game can be started.
